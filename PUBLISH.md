# Инструкция по публикации пакета в npm

## Подготовка

1. Убедитесь, что вы зарегистрированы на npm:
   ```bash
   npm login
   ```

2. Проверьте, что имя пакета доступно (если нужно изменить, отредактируйте `package.json`)

3. Заполните информацию в `package.json`:
   - `author` - ваше имя/email
   - `repository.url` - URL вашего репозитория (если есть)

## Сборка и публикация

1. Соберите проект:
   ```bash
   npm run build
   ```

2. Проверьте содержимое пакета (опционально):
   ```bash
   npm pack
   ```

3. Опубликуйте пакет:
   ```bash
   npm publish
   ```

   Для публикации публичного пакета:
   ```bash
   npm publish --access public
   ```

## Обновление версии

После внесения изменений обновите версию:

```bash
# Патч версия (1.0.0 -> 1.0.1)
npm version patch

# Минорная версия (1.0.0 -> 1.1.0)
npm version minor

# Мажорная версия (1.0.0 -> 2.0.0)
npm version major
```

Затем опубликуйте:
```bash
npm publish
```

## Проверка после публикации

После публикации проверьте, что пакет доступен:
```bash
npm view vite-plugin-dtsbom
```
